# Quick Start

You installed Regula. Here's what to do next.

## 1. Check your project

```bash
regula check /path/to/your/project
```

This scans your codebase and tells you which EU AI Act risk tier applies.

## 2. Don't have code? Use the questionnaire

```bash
regula assess
```

Answer yes/no questions about what your product does. No code required.

## 3. Get your compliance plan

```bash
regula plan /path/to/your/project
```

Prioritised list of what to fix, with effort estimates per article.

## 4. Generate documentation

```bash
regula docs /path/to/your/project
```

Auto-generates Annex IV technical documentation from your actual code.

## 5. Verify your installation

```bash
regula doctor
```

Runs 10 health checks to confirm everything is working.

---

Full command reference: `regula --help` or see the [README](../README.md).
